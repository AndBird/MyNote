# Git

## Git指令

```
//查看config配置
git config --list

//设置局部用户名和email
git config user.name  "username"      
git config user.email "email"     


//设置用户名和email
git config --global user.name  "username"
git config --global user.email "email"    


//清除缓存的用户名和密码
git credential-manager uninstall

//查看用户名
git config --global user.name


```
